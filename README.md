## Prerequisites

- Java 8 or higher

## Install

Add this dependency to your project's POM:

```xml
<dependency>
    <groupId>io.github.hezhangjian</groupId>
    <artifactId>collections</artifactId>
    <version>0.1.0</version>
</dependency>
```

## Support Data Structures

- **TrieTree**
